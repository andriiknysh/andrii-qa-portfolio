# Andrii QA Portfolio

## Evidence Policy

Real-world projects should include:

- Tested URL or app name
- Date tested
- Device, browser, and OS
- Test scope
- Screenshots or Loom recording when available
- Test cases
- Bug reports
- Summary and recommendations
