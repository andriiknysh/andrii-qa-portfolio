# Support Ticket Sample 2

## Customer Issue

User reports that an uploaded file appears stuck in processing.

## Clarifying Questions

- What file type and file size was uploaded?
- What time was the file uploaded?
- Does the issue happen with every file or only this one?
- Which browser and device were used?

## Troubleshooting Steps

1. Confirm file type and size.
2. Ask the user to refresh and check status again.
3. Try reproducing with a sample file.
4. Escalate with account ID, upload time, file type, and browser details.

## Suggested Response

I can help investigate the upload. Please share the file type, approximate file size, upload time, and browser used. I will use that information to check whether this is a temporary processing delay or a file-specific issue.
